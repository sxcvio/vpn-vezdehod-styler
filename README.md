## ⚡ Установка

### Вручную (уже сейчас)

1. Нажми **Code → Download ZIP** на этой странице
2. Распакуй архив
3. Открой Chrome: `chrome://extensions/`
4. Включи **«Режим разработчика»** (правый верхний угол)
5. Нажми **«Загрузить распакованное»** → выбери папку
6. Открой [acc.kompot.sbs](https://acc.kompot.sbs) — в шапке появится кнопка **Тема**

### Chrome Web Store

> Скоро — ссылка появится после публикации

## 🖱️ Использование

После установки на сайте появляется кнопка **Тема** рядом с **EN** и **Выйти**. Нажми — выбери тему. Выбор сохраняется автоматически.
