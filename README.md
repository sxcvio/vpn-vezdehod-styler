## О проекте

Устал от стандартного светло-серого дизайна VPN Вездехода? Это расширение добавляет кнопку **«Тема»** прямо в шапку сайта — рядом с EN и Выйти. Одним кликом переключай между 7 стилями без перезагрузки страницы. Тема сохраняется между сессиями.

## Установка

### Способ 1 — Chrome Web Store *(скоро)*

> Ссылка появится после публикации

### Способ 2 — Вручную

```bash
# 1. Скачай репозиторий
git clone https://github.com/YOUR_USERNAME/vpn-vezdehod-styler.git

# Или просто нажми Code → Download ZIP на этой странице
```

Затем:

1. Открой Chrome и перейди на `chrome://extensions/`
2. Включи **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажми **Загрузить распакованное**
4. Выбери папку с расширением
5. Готово — заходи на [acc.kompot.sbs](https://acc.kompot.sbs)

---

## Как пользоваться

После установки в шапке сайта появится кнопка **Тема** рядом с EN и Выйти. Нажми на неё — откроется выпадающее меню со всеми темами. Выбор сохраняется автоматически.

## Приватность

- Работает **только** на домене `acc.kompot.sbs`
- Не собирает никаких данных
- Не отправляет запросы на внешние серверы
- Выбранная тема хранится локально в `chrome.storage.local`
