# 🎨 VPN Vezdehod Styler

Расширение для Chrome, которое добавляет **7 красивых тем** на сайт [acc.kompot.sbs](https://acc.kompot.sbs). Переключение прямо из шапки сайта — одним кликом, без перезагрузки страницы.

## 🚀 Установка

### Вариант 1 — Через Chrome Web Store *(скоро)*
> Ссылка появится после публикации

### Вариант 2 — Вручную (уже сейчас)

1. **Скачай** репозиторий → кнопка `Code → Download ZIP` (или `git clone`)
2. Распакуй архив
3. Открой Chrome и перейди по адресу: `chrome://extensions/`
4. Включи **«Режим разработчика»** (правый верхний угол)
5. Нажми **«Загрузить распакованное»**
6. Выбери папку `vpn-styler-release`
7. Готово! Заходи на [acc.kompot.sbs](https://acc.kompot.sbs) — в шапке появится кнопка **Тема**

---

## 🖱️ Как пользоваться

После установки на сайте `acc.kompot.sbs` в шапке рядом с кнопками **EN** и **Выйти** появится кнопка **Тема**. Нажми на неё и выбери любой стиль из выпадающего меню. Тема сохраняется автоматически между визитами.

---

## 🛠️ Как обновить расширение

Если вышла новая версия:

1. Скачай новый ZIP с GitHub
2. Замени файлы в папке расширения
3. Зайди на `chrome://extensions/` → нажми кнопку 🔄 рядом с расширением

## 🔒 Приватность

Расширение работает **только** на `acc.kompot.sbs`. Не собирает никаких данных, не отправляет запросы на внешние серверы. Единственное хранилище — `chrome.storage.local` на твоём компьютере (для запоминания выбранной темы).
